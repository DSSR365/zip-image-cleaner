# 🚀 압축파일 내부 특정 이미지 일괄 제거 (Ad & XML Purifier)

**"대량의 압축파일 관리, 아직도 수작업으로 진행하고 계십니까?"**

본 프로그램은 다량의 압축파일(.zip, .cbz) 내부에 포함된 불필요한 이미지와 시스템 메타데이터(`ComicInfo.xml`)를 효율적으로 제거하기 위해 개발된 대량 처리 전용 유틸리티입니다. 수백, 수천 권 이상의 라이브러리를 보유한 헤비 소장러 및 업로더의 작업 시간을 획기적으로 단축해 드립니다.

사용 중 오류가 발생하거나 기능 개선 건의사항이 있으신 경우, 상단의 **[Issues]** 탭을 통해 상세히 등록해 주시면 감사하겠습니다.

프로그램 사용 중 버그나 건의사항이 있으시다면 상단의 [Issues] 탭에 글을 남겨주세요!

## ✨ 핵심 치트키 기능
1. **1000개 파일 일괄 스캔:** 파일이 아무리 많아도 압축을 풀지 않고 내부 구조를 순식간에 분석합니다.
2. **자릿수 무관 숫자 자동 매칭:** `01`, `001`, `00001` 등 자릿수가 뒤죽박죽이어도 순수한 번호 크기를 정확히 파악해 지정 범위와 매칭합니다.
3. **커스텀 단어 필터링:** 특정 단어(예: `image`, `ad`)의 앞/뒤 숫자를 내 맘대로 선택해 원하지 않는 광고 이미지만 칼같이 도려냅니다.
4. **완벽한 안전장치 (1초 복구):** 작업 전 자동으로 원본 백업본을 생성하며, 실수로 잘못 지웠더라도 주황색 버튼 클릭 한 번으로 모든 원본 파일을 즉시 되돌립니다.

## 🏃‍♂️ 사용 방법
1. 프로그램을 실행한 뒤, 만화 파일들이 모여있는 **대상 폴더**를 선택합니다.
2. 삭제하고 싶은 이미지의 **시작 번호와 끝 번호**를 지정합니다. (기본값 0~19)
3. 파일명에 맞는 **숫자 인식 방식**을 선택합니다. (전체 숫자 찾기 또는 특정 단어 기준 앞/뒤)
4. `🔍 조건에 맞는 파일 분석 시작`을 눌러 목록을 확인합니다.
5. `🗑️ 선택한 광고/XML 일괄 삭제 실행`을 누르면 끝!
